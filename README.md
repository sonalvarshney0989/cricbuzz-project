🏏 Cricbuzz LiveStats – Real-Time Cricket Insights & SQL-Based Analytics 📌 Project Overview

Cricbuzz LiveStats is a real-time cricket analytics dashboard that integrates live match data from the Cricbuzz API with a SQL database to deliver powerful analytics and interactive insights.

This project combines Python, SQL, Streamlit, REST API integration, and data analytics to build a fully functional multi-page web application focused on sports analytics.

🚀 Features ⚡ 1. Live Match Dashboard

Fetches real-time match data from Cricbuzz API

Displays:

Live scorecards
Batsman & bowler statistics
Match status -Venue details -Dynamic updates using API calls
📊 2. Top Player Statistics

Top run scorers

Highest individual scores
Most wickets
Format-wise performance insights
Clean visualization using Streamlit
🧮 3. SQL Analytics Module (25+ Queries)

Implemented Beginner, Intermediate & Advanced level SQL queries including:
Player performance analysis
Team win statistics
Venue-based insights
Toss impact analysis
Head-to-head comparison
Player consistency using standard deviation
Performance ranking system
Time-series player evolution tracking 🔄 4. Full CRUD Operations
Add new player records
Update existing data
Delete records
Read and filter database entries
Form-based UI built with Streamlit
